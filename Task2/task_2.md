# Завдання 2: Отримання інформації про котів

## Умова

У вас є текстовий файл, який містить інформацію про **котів**.
Кожен рядок файлу містить **унікальний ідентифікатор**, **ім'я** та **вік**, розділені **комами**.

### Приклад вмісту файлу:

```
60b90c1c13067a15887e1ae1,Tayson,3
60b90c2413067a15887e1ae2,Vika,1
60b90c2e13067a15887e1ae3,Barsik,2
60b90c3b13067a15887e1ae4,Simon,12
60b90c4613067a15887e1ae5,Tessi,5
```

## Завдання

Розробіть функцію `get_cats_info(path)`, яка:

* читає файл,
* повертає **список словників** з інформацією про кожного кота.

### Параметри функції

* `path` — шлях до текстового файлу з даними.

### Очікуваний результат

Функція повертає список такого вигляду:

```python
[
    {"id": "60b90c1c13067a15887e1ae1", "name": "Tayson", "age": "3"},
    {"id": "60b90c2413067a15887e1ae2", "name": "Vika", "age": "1"},
    {"id": "60b90c2e13067a15887e1ae3", "name": "Barsik", "age": "2"},
    {"id": "60b90c3b13067a15887e1ae4", "name": "Simon", "age": "12"},
    {"id": "60b90c4613067a15887e1ae5", "name": "Tessi", "age": "5"},
]
```

## Вимоги до реалізації

* Використовуйте менеджер контексту `with` для читання файлів.
* Встановіть **коректне кодування** при відкритті файлу (`encoding="utf-8"`).
* Для кожного рядка застосовуйте `.split(',')` для отримання `id`, `name`, `age`.
* Створіть словник з ключами `"id"`, `"name"`, `"age"`.
* Додайте кожен словник у список, який буде повернуто.
* Обробляйте винятки, пов’язані з читанням файлу.

## Критерії оцінювання

* Функція повинна коректно обробляти дані і повертати точний список словників.
* Повинна бути реалізована належна обробка винятків.
* Код має бути чистим, структурованим і зрозумілим.

## Приклад використання

```python
cats_info = get_cats_info("path/to/cats_file.txt")
print(cats_info)
```
